NoFox's
RGB
Guessing Game

This is a small adaptation of one of the exercises for the Web Developer Boot Camp course I completed in late 2017. Though mostly review for about the first third of the course it served to both strengthen my foundation in modern, mobile first web design as well as providing a few updates to outdated practices.

The game, though simple, allows a quick display of such skills as Vanilla JavaScript DOM manipulation, additional JavaScript logic, linking multiple CSS files for flexibility, and adapting codes and concepts for my own use. Though similar, I have made several small updates to the applet to make it "mine".
